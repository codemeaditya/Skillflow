1. What HTML Really Is

HTML is not a programming language. It is a **markup language** used to describe the structure and meaning of content on the web. Browsers read HTML and build the DOM (Document Object Model).

2. HTML Document Anatomy

  • <!DOCTYPE html> — tells browser to use HTML5
  • <html> — root element
  • <head> — metadata, SEO, fonts, CSS
  • <body> — visible content
